# Task 006: Frontend Team Page

## Description
Build the `TeamPage` component with hero section (gradient background, logo, name, record), team description (collapsible, 2-line default), and a link placeholder for tactics.

## Acceptance Criteria
- Hero section styled with MLB Navy gradient (`#041E42` -> `#0A2A5A`)
- Logo displayed (120px) with fallback placeholder if `logo_url` missing
- Team name in white bold 28px+
- Record displayed with wins/losses and win rate; numbers use MLB Red (`#BF0D3E`)
- Description collapsible with expand/collapse control

## Technical Details
- Use Tailwind CSS for layout and colors
- Fetch data from `GET /api/teams/:id`
- Handle loading and error states explicitly

## Dependencies
- Task 005 (routing)
- Task 003 (backend API) for integration

## Effort Estimate
0.5 day

## Definition of Done
- Component renders correctly with sample API responses
- Unit tests for loading/error/success states
