# Task 007: Frontend Member Grid

## Description
Build the `MemberGrid` component that displays team members in a responsive grid with avatar, name, jersey number, position, and a captain gold badge.

## Acceptance Criteria
- Grid is 4 columns on desktop, 2 columns on mobile
- Each card shows circular 56px avatar with fallback placeholder
- Displays name, jersey number, and position
- Captain has a gold (`#C4A35A`) crown/star indicator
- Online status indicator shown with green dot (`#2D8659`)

## Technical Details
- Fetch data from `GET /api/teams/:id/members`
- Use Tailwind grid utilities (`grid-cols-2 md:grid-cols-4`)
- Cards have white background and subtle shadow per spec

## Dependencies
- Task 006 (TeamPage shell)
- Task 003 (backend API) for integration

## Effort Estimate
0.5 day

## Definition of Done
- Component renders with fetched member data
- Responsive layout verified via tests or manual resize
