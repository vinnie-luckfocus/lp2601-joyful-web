# Task 005: Lineup List Component with Current-User Highlight

## Description

Build the `LineupList` component that displays the 1-9 batting order. Highlight the row corresponding to the currently logged-in user with a distinct red background.

## Acceptance Criteria

- [ ] Renders 1-9 rows showing `batting_order`, `name`, `jersey_number`, and `position`
- [ ] Current user's row uses MLB Red (`#BF0D3E`) background highlight
- [ ] List background uses MLB Navy (`#041E42`) with white text
- [ ] Rows are evenly spaced and scrollable if content overflows
- [ ] Component accepts `lineup` and `currentUserId` props

## Technical Details

- React + TypeScript
- Tailwind CSS for layout and colors
- Use `currentUserId` from auth store (Zustand) to determine highlight
- Subtle pulse or glow animation on current-user row (CSS keyframes)

## Dependencies

- Existing auth store pattern

## Effort Estimate

2-3 hours

## Definition of Done

- Component file created
- Unit tests verify highlight logic
- Visual inspection confirms colors and spacing
