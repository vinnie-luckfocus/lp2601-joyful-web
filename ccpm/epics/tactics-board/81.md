# Task 006: Tactics Panel Component

## Description

Build the `TacticsPanel` component that displays collapsible cards for general notes, signals, and defensive strategy. Cards should animate expand/collapse with a height transition.

## Acceptance Criteria

- [ ] Three collapsible sections: General Notes, Signals, Defense Strategy
- [ ] Each section shows a title and expand/collapse toggle
- [ ] Expanded content renders inside a white card with a 4px left border in `#BF0D3E`
- [ ] Height animation on expand/collapse (300ms)
- [ ] Signals section gracefully renders JSON key/value pairs as a simple list
- [ ] Component accepts a `tactics` prop

## Technical Details

- React + TypeScript
- Tailwind CSS for cards and transitions
- CSS `transition-all` or Framer Motion for height animation
- Keep animations performant (use `transform` or `max-height`)

## Dependencies

- None (pure UI component)

## Effort Estimate

2-3 hours

## Definition of Done

- Component file created
- Unit tests for expand/collapse behavior pass
- Animation is smooth on desktop and mobile
