# Task 008: Responsive Layout and Animations

## Description

Polish the tactics board page with responsive layout and animations. Desktop shows lineup list and field diagram side-by-side; mobile stacks them vertically. Add staggered load animations for field markers.

## Acceptance Criteria

- [ ] Desktop: lineup list 40% width, field diagram 60% width, side-by-side
- [ ] Mobile: stacked vertically, lineup list on top, full width
- [ ] Field diagram position markers animate in with a stagger of ~80ms each
- [ ] Tactics panel stacks below the main content on both breakpoints
- [ ] No layout shift or horizontal scroll on typical mobile screen widths
- [ ] Current-user highlight in lineup list has a subtle repeating pulse (CSS)

## Technical Details

- Tailwind CSS grid/flex and responsive breakpoints (`md:`, `lg:`)
- CSS `@keyframes` for marker pop-in and staggered delay via inline `animation-delay`
- Prefer CSS animations over JS to keep bundle lean

## Dependencies

- Task 007 (page component)

## Effort Estimate

2-3 hours

## Definition of Done

- Responsive layout verified at 375px, 768px, and 1440px widths
- Animations run at 60fps
- Page passes manual UX review
