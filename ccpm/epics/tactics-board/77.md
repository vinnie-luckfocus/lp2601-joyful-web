# Task 002: Seed Data for Lineup and Tactics

## Description

Create seed data for local development and E2E tests. Seed at least one game with a complete 9-player lineup and associated tactics (general notes, signals, defense strategy).

## Acceptance Criteria

- [ ] Seed script inserts 9 lineup records for a test game with `batting_order` 1-9
- [ ] Each lineup record links to an existing test user and includes `position` and `jersey_number`
- [ ] Seed script inserts one `game_tactics` record with `general_notes`, `signals`, and `defense_strategy`
- [ ] Seed runs idempotently (can be re-run without duplicates)

## Technical Details

- Use existing seed pattern in `database/seeds/`
- Reference existing test users from auth/user seeds
- Include diverse defensive positions (P, C, 1B, 2B, 3B, SS, LF, CF, RF)
