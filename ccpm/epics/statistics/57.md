# Task 008: Frontend — Milestones and Leaderboard Section

## Description

Add the milestones strip and public leaderboard section to the player stats page.

## Acceptance Criteria

- [ ] Milestones display as horizontal cards at the bottom (e.g., "Distance to 100 hits: 23")
