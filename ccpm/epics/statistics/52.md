# Task 006: Frontend — Player Stats Dashboard (Profile + Cumulative Cards)

## Description

Create the player stats page with the top profile header and a responsive grid of cumulative stat cards.

## Acceptance Criteria

- [ ] Route `/stats/me` renders profile header (avatar, name, team, jersey number)
- [ ] Cumulative stat cards display: games, at_bats, hits, batting_avg, singles, doubles, triples, hr, rbi, runs, walks, strikeouts
- [ ] Cards use MLB Navy for labels and MLB Red for key numbers (batting_avg)
- [ ] Responsive grid: 3 columns desktop, 2 tablet, 1 mobile
- [ ] Numbers animate from 0 to final value on page load (1000ms easeOut)

## Technical Details

- Use Tailwind CSS for layout and responsive grid
- Reuse existing avatar component if available
- Animate numbers via requestAnimationFrame or a lightweight custom hook
- Match PRD color palette exactly

## Dependencies

- Task 003 (stats_me_api)

## Effort Estimate

5 hours

## Definition of Done

- Page renders correctly with seeded data
- Animations smooth at 60fps
- Responsive breakpoints verified
