# Task 007: Frontend — Player Stats Charts (Trend + Radar)

## Description

Add the trend line chart and five-dimension radar chart to the player stats page using Recharts.

## Acceptance Criteria

- [ ] Trend chart shows last 5 games batting average as a line chart
- [ ] Trend line draws left-to-right with an 800ms animation
- [ ] Radar chart displays 5 dimensions: hitting, fielding, baserunning, power, speed
- [ ] Radar vertices connect clockwise with a 600ms draw animation
- [ ] Charts are responsive and fit the 60/40 split layout on desktop

## Technical Details

- Install `recharts` if not already present
- Trend chart: `LineChart` with single series, filled area under line
- Radar chart: `RadarChart` with 5 axes; normalize values to 0-100 scale
- Color specs from PRD: line `#041E42`, fill `rgba(4,30,66,0.1)`, radar fill `rgba(4,30,66,0.3)`, vertices `#BF0D3E`

## Dependencies

- Task 003 (stats_me_api)
- Can be developed in parallel with Task 006 as long as they touch different files

## Effort Estimate

6 hours

## Definition of Done

- Both charts render correctly with seeded data
- Animations match timing specs
- Responsive layout tested on mobile and desktop
