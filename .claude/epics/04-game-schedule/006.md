# Task: Build frontend hooks and GameSchedulePage

## Description

Create custom React hooks for games and attendance data, then build the `GameSchedulePage` with a month filter, timeline layout, and MLB-style stagger card animations.

## Acceptance Criteria

- [ ] `useGames()` hook fetches and caches game schedule data (public or authenticated as needed)
- [ ] `useAttendance(gameId)` hook manages optimistic local state and the attend/decline mutation
- [ ] `GameSchedulePage` displays games in a timeline layout: time on the left, card on the right
- [ ] Page background uses `#F5F7FA` per PRD MLB style
- [ ] Month switcher at the top allows filtering by month with left/right arrows
- [ ] Switching months triggers a stagger slide-in animation (50ms delay between cards)
- [ ] Skeleton loader uses MLB Navy (`#041E42`) tone while data loads
- [ ] Page is reachable via the application router at `/schedule`

## Technical Details

- Use existing axios utils in `frontend/src/utils/axios.ts`
- Use Tailwind CSS for layout and transitions
- Use `framer-motion` if already in the project, otherwise Tailwind keyframes for stagger animation
- Month filter is client-side only after the initial fetch
- "实时性" is achieved via optimistic UI for the acting user; other users see updates on next page load
