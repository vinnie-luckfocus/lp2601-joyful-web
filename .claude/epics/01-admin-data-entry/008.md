# Task: Admin Layout Framework

## Description
创建管理后台布局框架，包括侧边栏、头部和内容区组件。

## Acceptance Criteria
- [ ] 创建 Sidebar 组件（固定 240px，MLB Navy 背景）
- [ ] 创建 Header 组件（顶部导航，用户信息）
- [ ] 创建 ContentArea 组件（可滚动内容区）
- [ ] 实现响应式布局（移动端侧边栏可收起）
- [ ] 侧边栏菜单项高亮效果（MLB Red 左边框）
- [ ] 头部显示当前用户和登出按钮
- [ ] 布局组件样式符合 MLB 主题

## Technical Details

### 组件结构
```typescript
// components/Layout/Sidebar.tsx
interface SidebarProps {
  collapsed?: boolean;
  onCollapse?: (collapsed: boolean) => void;
}

// components/Layout/Header.tsx
interface HeaderProps {
  user: {
    name: string;
    avatar?: string;
  };
  onLogout: () => void;
}

// components/Layout/ContentArea.tsx
interface ContentAreaProps {
  children: React.ReactNode;
}

// components/Layout/index.tsx
export const AdminLayout: React.FC = ({ children }) => {
  return (
    <div className="flex h-screen">
      <Sidebar />
      <div className="flex-1 flex flex-col ml-60">
        <Header />
        <ContentArea>{children}</ContentArea>
      </div>
    </div>
  );
};
```

### 样式规范
- 侧边栏: 固定宽度 240px，高度 100vh，背景色 MLB Navy `#041E42`
- 选中项: 左边框 4px MLB Red `#BF0D3E`，背景色半透明
- 头部: 高度 64px，白色背景，底部阴影
- 内容区: 背景色 `#F5F7FA`，内边距 24px
