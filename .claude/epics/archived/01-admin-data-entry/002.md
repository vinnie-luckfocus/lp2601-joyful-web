# Task: Database Seed Data

## Description
创建种子数据脚本，包含 2 个球队、每队 9+ 队员、4 周赛程，用于开发和测试。

## Acceptance Criteria
- [ ] 创建 2 个球队（A队、B队）
- [ ] 每队至少 9 名队员
- [ ] 队员包含完整信息（姓名、背号、位置）
- [ ] 创建 4 周赛程（每周 1-2 场比赛）
- [ ] 创建 1 个管理员账号
- [ ] 种子数据脚本可重复执行（支持清空后重建）
- [ ] 数据符合真实联赛场景

## Technical Details

### 种子数据结构
```javascript
// teams
const teams = [
  { name: 'Joyful A队', division: '大联盟' },
  { name: 'Joyful B队', division: '大联盟' }
]

// players (每队 9+)
const players = [
  { name: '张三', jersey_number: 1, position: '投手', team_id: 1 },
  { name: '李四', jersey_number: 2, position: '捕手', team_id: 1 },
  // ... 每队 9+
]

// games (4周)
const games = [
  { scheduled_at: '2025-04-12 14:00', location: 'A球场', home_team_id: 1, away_team_id: 2 },
  { scheduled_at: '2025-04-19 14:00', location: 'B球场', home_team_id: 2, away_team_id: 1 },
  // ... 4周赛程
]

// admin user
const admin = {
  username: 'admin',
  password_hash: '...',
  name: '管理员',
  role: 'admin'
}
```

### 实现方法
1. 创建 database/seeds/ 目录
2. 编写 seed.js 或 seed.sql 脚本
3. 使用 bcrypt 生成管理员密码哈希
4. 脚本支持幂等执行（先清空再插入）
