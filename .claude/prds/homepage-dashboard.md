# PRD: Homepage Dashboard

## Executive Summary

首页看板是 Joyful 棒球联赛网站的门面，面向所有访客（包括未登录用户）展示联赛的公开信息。通过直观的赛程卡片、实时积分榜、数据排行榜和最近战报，吸引潜在队员关注并引导现有队员登录。

## Problem Statement

- **问题**：潜在队员想了解联赛情况但没有入口；现有队员每次查看信息都需要登录
- **重要性**：降低信息获取门槛，提升网站活跃度，展示联赛专业性

## User Stories

### 故事1：潜在队员了解联赛
**作为** 对棒球感兴趣的人
**我希望** 在不注册的情况下看到联赛赛程和水平
**从而** 判断是否适合加入

**验收标准**：
- [ ] 无需登录即可访问首页
- [ ] 能看到未来4周的比赛安排
- [ ] 能看到各球队的战绩排名
- [ ] 能看到顶尖选手的数据表现

### 故事2：队员快速查看信息
**作为** 已注册的队员
**我希望** 不用登录就能快速扫一眼近期赛程
**从而** 决定是否值得登录查看详情和报名

**验收标准**：
- [ ] 首页加载速度快（<2秒）
- [ ] 赛程信息一目了然
- [ ] 有明显的登录入口

## Requirements

### Functional Requirements

| 功能 | 描述 | 优先级 |
| :--- | :--- | :--- |
| 英雄区 | 展示联赛名称、口号、登录按钮 | P0 |
| 赛程卡片 | 展示未来4场比赛（时间、场地、对阵） | P0 |
| 积分榜 | 展示所有球队排名（胜/负/胜率） | P0 |
| 数据排行榜 | 展示打击率、安打、本垒打、打点Top10 | P0 |
| 最近战报 | 展示最近3场比赛的比分和亮点 | P1 |
| 登录入口 | 顶部导航栏"队员登录"按钮 | P0 |
| 查看更多 | 各模块跳转到完整页面的链接 | P1 |

### API Requirements

```
GET /api/public/games?limit=4
- 返回近期4场比赛信息
- 公开访问，无需认证

GET /api/public/standings
- 返回球队积分榜
- 包含：球队名、胜场、负场、胜率

GET /api/public/leaders?category=[batting_avg|hits|hr|rbi]&limit=10
- 返回各类数据排行榜
- 支持分类筛选

GET /api/public/recent-games?limit=3
- 返回最近3场比赛结果
- 包含比分和简要数据亮点
```

### Non-Functional Requirements

| 类型 | 要求 |
| :--- | :--- |
| 性能 | 首屏加载 < 2秒，骨架屏过渡 |
| SEO | 基础SEO优化，可被搜索引擎收录 |
| 响应式 | 完美适配移动端和桌面端 |
| 缓存 | 公开数据可CDN缓存5分钟 |

## Success Criteria

| 指标 | 目标值 | 测量方式 |
| :--- | :--- | :--- |
| 首页访问量 | 每周100+ UV | Google Analytics |
| 登录转化率 | 20%访客点击登录 | 埋点统计 |
| 页面停留时间 | 平均30秒+ | Google Analytics |

## Constraints & Assumptions

- 公开数据不包含敏感信息（如队员联系方式）
- 数据更新有5分钟延迟（缓存）
- 假设联赛有稳定的数据来源

## Out of Scope

- 用户注册功能（在会员系统PRD中）
- 实时数据推送（轮询即可）
- 复杂的筛选和搜索功能

## Dependencies

- 依赖后端公开API接口
- 依赖基础数据库表（games, teams, users, batting_records）
